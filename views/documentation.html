User manual

Projektet består av två separata webbservrar: en javaserver skriven med ramverket Spark Java [1] som används för API:et och en server skriven i Python med ramverket Bottle [2] som levererar html. Klona projektet från https://github.com/annimiriam/WebbProjektQuiz.git

Python-servern startas genom att köra WebbProjektQuiz/main.py. Detta startar en webbserver på localhost:8083. Bottle kräver inga externa dependencies utöver bottle.py som levereras tillsammans med projektet. Javaservern är ett mavenprojekt och Maven måste därför installeras. Se [3] om instruktion för detta önskas. Javaservern använder API-nycklar för att kunna använda API från Youtube och quizapi.io. Nycklarna är lagrade i miljövariablerna ‘youtube’ och ‘quiz’ och måste sparas i dessa innan servern kan startas.

Nyckel för variabeln youtube: AIzaSyDwxtKj6J7Zb0vYqUJ5Uxc4usz5ZWbtUrE Nyckel för variabeln quiz: FuHvfV9ziQUPHthjIIojiW7APZcWdh4AVaYTUseG

För att köra java-servern: Navigera till WebbProjektQuiz/server. Kör kommandot ‘mvn compile’. Kör kommandot ‘mvn exec:java -Dexec.mainClass=Main’ för att starta servern på localhost:4567. Öppna http://localhost:8083 i en webbläsare.

Documentation

Quizmanias API ger tillgång till quiz om flertalet kategorier relaterade till systemutveckling och informationsarkitektur, samt video-id:n för att enkelt kunna använda till inbäddade Youtube-videor kring samma kategori. API:et följer RESTs principer. Anrop till ändpunkten categories returnerar de tillgängliga kategorierna UTVECKLA MER Dokumentationen finns även tillgänglig under fliken Documentation på QuizManias webbsida.

Headers

HTTP-headern Accept: application/json kan användas. Returdatan representeras i JSON. GET /api/categories Returnerar de kategorier som API:et erbjuder. Exempel på returdata:

GET /api/categories/{category}

En viss frågekategori nås med hjälp av ett kategorinamn. De skrivs nedan som {category}. Ersätt därför hela {category} med kategorinamnet. Exempel på kategorinamn: html. Ger en lista med frågor, svarsalternativ, information om det finns fler än ett korrekt svar, de korrekta svaren och video-id:n. För returdatan nedan har kategoriparametern html använts och parametern nbrOfQuestions har satts till 1. Exempel på returdata:

Parametrar

nbrOfQuestions

Bestämmer hur många quizfrågor som returneras. Kan vara mellan 1-20. Om inget anges så returneras 20 frågor.

nbrOfVideos

Bestämmer hur många video-id:n som returneras. Kan vara mellan 1-20. Om inget anges så returneras 5 video-id:n.

Exempel: http://localhost:4567/api/categories/html?nbrOfQuestions=3&nbrOfVideos=2 API:et returnerar ett svar med 3 frågor och 2 matchande videolänkar.
